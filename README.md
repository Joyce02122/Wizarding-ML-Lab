## Wizarding ML Lab

Inspired by the Harry Potter universe, this project brings magical experiences into the real world through embedded systems and machine learning.

### 🎩 Sorting Hat

A Hogwarts house assignment system powered by a **Decision Tree classifier**. Users answer a series of questions through physical button inputs, and the system predicts their Hogwarts house based on their responses.

### 🪄 Magic Wand

A gesture-recognition system that transforms a physical wand into a spell-casting device. Using motion sensor data and machine learning, the wand recognizes predefined gestures such as circles, zigzags, and check marks in real time.

### 🚀 Project Goal

This project explores how embedded systems, physical computing, and machine learning can transform fictional magic into tangible interactive experiences.

### 🛠️ Technologies

- Machine Learning
  - Decision Tree Classification
  - Gesture Recognition
- Embedded Systems
- Physical Computing
- Sensor Data Processing
- Arduino / Microcontroller Development
